# Chatbot SYNECTIC

Este proyecto está relacionado con el desarrollo de un asistente virtual de resolución de incidencias informáticas para la empresa SYNECTIC. Los incidencias que se atienden forman parte de las siguientes categorías:
* APLICACIONES (incluye Office365, Windows, SAP...).
* CARPETAS COMPARTIDAS.
* CREDENCIALES.
* REDES.
* GESTIÓN SOFTWARE/HARDWARE.
